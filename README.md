# Tawhid-Vedanta Bridge

[![Version](https://img.shields.io/badge/version-1.0-blue.svg)](https://github.com/yourusername/tawhid-vedanta-bridge) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

The Tawhid-Vedanta Bridge: Comparative educational framework linking Islamic Tawhid & Sufi mysticism with Advaita Vedanta & Bhakti. Designed for ex-Muslims and monotheistic seekers exploring Sanatana Dharma. Addresses 100 common objections non-proselytizingly, highlighting parallels in non-dualism, ethics & religious transition. Open-source reckoner.

## Live Website

https://yourusername.github.io/tawhid-vedanta-bridge

(The site is styled in **dark mode** by default with **vibrant accent buttons** for better readability and modern feel.)

## Features

- Single-page React application with smooth section navigation
- Collapsible objection categories for focused reading
- Mobile-responsive design with hamburger menu
- Dark theme with high-contrast text and vibrant blue/green accent buttons
- Uses Lucide React icons for clean, modern UI
- Non-proselytizing, anonymity-safe educational content

## Sections

- Title & Abstract
- Literature Review
- Methodology
- Conceptual Framework
- Addressing Objections (100 common concerns categorized)
- Discussion (partial in current version)
